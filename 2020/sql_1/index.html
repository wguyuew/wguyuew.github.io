MySql操作

数据库基础

  • 数据库：保存有组织的数据的容器。

  • 表：某种特定类型结构化清单。

  • 模式:关于数据库和表的布局及特性的信息。

  • 列:表中的一个字段，所有表都是从一个或多个列组成的。

  • 行:表中的一个记录

  • 主键(primary key):一列(一组列)其值能唯一区分表中的每个行

    • 主键必须满足的条件: 任意两行都不具有相同的主键值； 每个行都必须具有一个主键值（主键列不允许NULL值）。
    • 主键最好满足的条件: 不更新主键列中的值； 不重用主键列的值； 不在主键列中使用可能会更改的值;

检索数据

  • 检查单个列(未排序)
select  Sno(检查的列)
from Student(列所在的表);

  • 检查多个列
select Sno,Sname,Ssex
from Student;

  • 检查所有的列
select *
form Student;

  • 检索不同的行 重复元素
select  Sno
from Sc;

无重复元素

select distinct Sno
from Sc;

  • 限制结果
select Sno
from Student
limit 3;//限制的个数


select Sno
from Student
limit 2,3;

  • 使用完全限定的名称
select Student.Sno
from Student

排序检索数据

  • 排序数据
  • 无order by
select Sno
from Student;

  • order by(对查询结果按照一个或多个属性列升序(ASE)或降序(DASE)，默认升序)
select Sno
from Student
order by Sno;

  • 按多个列排序
selsect Sno,Sname,Sage
from Student
order by Sage;

过滤数据

where语句

  • 检查单个值
select Sno
from Studnet
where Sname = '张三';


select Grade 
from Sc
where Grade<80;

  • 不匹配查询
select Sname
from Stduent
where Sname <> '张三';//可将<>换位！=

  • 范围值检查
select Grade
from Sc
where Grade between 5 and 80;

  • 空值检查
    where xxx is null;

数据过滤

  • and

    select Sno,Grade
from Sc
where Cno = '002' and Grade >10;

  • or

    select Sname
from Student
where Sage = '18' or Sge = '19';

    and运算次序比or高

  • in

select Sno,Grade
from Sc
where Cno in (001,002);

  • not
select Sno,Grade
from Sc
where Cno not in (001,002);

使用子查询

select Sno
from Student
where Sno in (
    select Sno
    from Sc
    where Grade > 90
);
